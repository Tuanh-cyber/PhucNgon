# Therapy Rules

## Purpose

This document defines the therapy flow for the Logic Sequence exercise.

The exercise trains patients to recognize the correct order of daily activities by arranging images into their logical sequence.

---

# Therapy Flow

Each exercise follows the workflow below:

Play Instruction Audio
        ↓
Load Sequence
        ↓
Shuffle Images
        ↓
Patient Arranges Images
        ↓
Submit
        ↓
Receive Score
        ↓
Retry or Continue

---

# Exercise Rules

The Logic Sequence exercise contains three difficulty levels.

## Level 1

- 3 image steps
- Familiar daily activities

Examples

- Trồng cây
- Ăn bánh
- Thay đồ
- Dọn cỏ
- Người tuyết

---

## Level 2

- 4 image steps
- Longer activities

Examples

- Trồng táo
- Chăm cún
- Đi học

---

## Level 3

- 5 image steps
- More complex daily activities

Examples

- Nấu ăn
- Giặt quần áo
- Đi bệnh viện

---

# Scoring Rules

Following the scoring rules for non-weight exercises.

# Session Rules

Each therapy session consists of 10 exercises.

The patient may

- Retry the current exercise
- Continue to the next exercise

The recommended session duration is 30 minutes.

During each session, the system records

- Patient profile
- Exercise type
- Sequence ID
- Difficulty level
- Exercise score
- Completion status
- Retry count
- Session duration
