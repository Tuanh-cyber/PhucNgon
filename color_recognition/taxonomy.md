## Exercise Information

| Attribute | Value |
|-----------|-------|
| Exercise Type | Color Recognition |
| Interaction | Touch Selection |
| Input Modalities | Audio + Visual |
| Output Modalities | Touch |
| Difficulty Levels | Level 1 |
| Target Profiles | Broca, Wernicke, Mixed Aphasia |

---

# Exercise Objective

The Color Recognition exercise evaluates the patient's ability to:

- Understand simple verbal instructions.
- Recognize and distinguish basic colors.
- Associate spoken color names with their corresponding visual representations.
- Improve auditory comprehension and visual perception.
